Translation Evaluator

Level

A1 A2 B1 B2 C1 C2

This application is a web-based tool that allows users to perform text translations in different languages and evaluate the quality and accuracy of translations. Users can input both original and translated texts to assess the translation. Additionally, users can select their language proficiency levels.

Key Components

  • Level: Allows users to determine their language proficiency levels.
  • Language Selection: Choose one of the provided languages to perform translations.
  • Text Fields: Enter original and translated text to evaluate the translation. You don't need to randomly generate original text, you can also provide text manually.
  • Generate a sentence: Generates a random sentence.
  • Evaluate the sentence: Evaluates the translation of the generated sentence.

Usage

  • Select Language Proficiency Level: Use the proficiency slider to set your level. (you cannot choose the same two languages)
  • Enter Text: Enter the original text and translation, select languages.
  • Use Buttons: Generate a random sentence and evaluate the translation.

This application is an effective tool for language learners and translators, offering a user-friendly interface and functionalities.
